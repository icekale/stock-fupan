# 开发与测试

## 后端开发

```bash
cd apps/api
uv sync
PYTHONPATH=. .venv/bin/python -m uvicorn app.main:app --reload --port 8000
```

API 地址：

```text
http://localhost:8000
```

## 前端开发

仓库根目录：

```bash
corepack enable
pnpm install
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000 pnpm dev:web
```

如果 `pnpm` shim 不可用：

```bash
corepack pnpm install
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000 corepack pnpm --filter @stock-review/web dev
```

前端地址：

```text
http://localhost:3000
```

## 常用测试命令

后端全量测试：

```bash
cd apps/api
PYTHONPATH=. .venv/bin/python -m pytest -q
```

Lint：

```bash
cd apps/api
PYTHONPATH=. .venv/bin/python -m ruff check app tests
```

生成报告：

```bash
make report DATE=2026-05-27
```

Docker build：

```bash
docker build -t stock-review-api:local apps/api
```

## 代码风格

- 小步改动，避免无关重构。
- 生产报告路径不能偷偷回落到 fake 数据。
- 涉及报告内容变更时，要补测试锁定 HTML 或结构化 DTO 行为。
- 涉及 provider 行为时，要检查 `provider_status`。
