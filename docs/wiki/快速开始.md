# 快速开始

本页介绍如何在本地运行项目并生成每日复盘报告。

## 1. 克隆项目

```bash
git clone https://github.com/icekale/stock-fupan.git
cd stock-fupan
```

## 2. 后端依赖

```bash
cd apps/api
uv sync
uv run playwright install chromium
```

如果本机没有 `uv`，先安装 `uv`；项目后端使用 Python 3.12。

## 3. 配置环境变量

在 `apps/api/.env` 中配置真实数据源：

```dotenv
MARKET_PROVIDER=tickflow
NEWS_PROVIDER=anspire
TICKFLOW_PROVIDER=tickflow
PROVIDER_FALLBACK_ENABLED=false
REVIEW_SOURCES_ENABLED=true

TICKFLOW_API_KEY=你的 TickFlow key
ANSPIRE_API_KEY=你的 Anspire key
```

注意：不要把 `.env` 提交到 GitHub。仓库已通过 `.gitignore` 忽略 `.env`。

## 4. 生成每日报告

回到仓库根目录：

```bash
make report DATE=2026-05-27
```

命令成功后会打印：

- `HTML`: 生成的 HTML 报告路径。
- `Snapshot`: 生成的快照 JSON 路径。
- `Validation`: 报告校验状态。
- `Provider ...`: 数据源状态。

## 5. 预览 HTML

进入生成目录，例如：

```bash
cd reports/2026-05-27/close/v003
python3 -m http.server 8890 --bind 127.0.0.1
```

然后浏览器打开：

```text
http://127.0.0.1:8890/report.html
```

## 6. 判断是否是正式可用报告

正式报告至少应满足：

- `Provider market: success`
- `fallback_used=false`
- 同花顺/东方财富复盘源尽量成功。
- 报告中强势板块来自当天真实行情，而不是 fake 示例数据。

如果出现 fallback，不建议把该报告当作正式复盘使用。
