# 数据源与环境变量

项目当前以真实数据为优先级，生产级报告不允许混入 fake 内容。

## 行情：TickFlow

当前行情源已经统一为 TickFlow，不再使用 AkShare。

```dotenv
MARKET_PROVIDER=tickflow
TICKFLOW_API_KEY=你的 TickFlow key
TICKFLOW_BASE_URL=https://api.tickflow.org
PROVIDER_FALLBACK_ENABLED=false
```

说明：

- `MARKET_PROVIDER=tickflow` 是默认真实行情路径。
- `PROVIDER_FALLBACK_ENABLED=false` 表示行情失败时直接失败，不生成 fake 混合报告。
- `snapshot.json` 中的 `provider_status.market` 可用于确认是否使用真实行情。

## 新闻：Anspire

```dotenv
NEWS_PROVIDER=anspire
ANSPIRE_API_KEY=你的 Anspire key
ANSPIRE_BASE_URL=https://plugin.anspire.cn/api/ntsearch/search
NEWS_TOP_K=10
NEWS_LOOKBACK_HOURS=36
```

说明：

- Anspire 用于搜索强势板块相关新闻。
- 新闻搜索失败会体现在 provider diagnostics 中。
- 生产报告不应依赖 fake 新闻。

## 复盘源：同花顺 + 东方财富

```dotenv
REVIEW_SOURCES_ENABLED=true
THS_FUPAN_URL=https://stock.10jqka.com.cn/fupan/
EASTMONEY_ZTFP_URL=https://stock.eastmoney.com/a/cztfp.html
```

用途：

- 确认强势主题是否被主流复盘源提到。
- 提取热点股票、板块主题、市场备注。
- 辅助生成“次日强势概率与观察条件”。

## 结构化复盘：规则 / LLM

默认本地规则模式：

```dotenv
LLM_PROVIDER=fake
STRUCTURED_REVIEW_PROVIDER=rule
```

可选 LLM 模式：

```dotenv
LLM_PROVIDER=openai
STRUCTURED_REVIEW_PROVIDER=llm
STRUCTURED_REVIEW_FALLBACK_ENABLED=true
OPENAI_API_KEY=你的 OpenAI-compatible key
OPENAI_BASE_URL=https://api.openai.com/v1
LLM_MODEL=gpt-4.1-mini
```

建议：MVP 阶段优先保持规则模式稳定，避免 LLM 生成不可控内容。

## OCR 自选股导入

```dotenv
OCR_PROVIDER=fake
OCR_FALLBACK_ENABLED=true
OCR_MODEL=gpt-4.1-mini
```

当 `OCR_PROVIDER=openai` 时，截图自选股导入会调用 OpenAI-compatible 视觉模型。

## 自选股模块

默认关闭：

```dotenv
REPORT_WATCHLIST_ENABLED=false
WATCHLIST_PROVIDER=local
WATCHLIST_SNAPSHOT_ROOT=./data/watchlists
```

开启后：

```dotenv
REPORT_WATCHLIST_ENABLED=true
```

开启后报告会增加“自选股观察”模块，并调用 TickFlow 获取自选股行情。
