# 路线图

当前项目已经完成本地 MVP 的关键骨架：真实行情、真实新闻、复盘源、结构化报告、参考风格 HTML、GitHub 发布。

## 优先级 1：报告质量继续提升

- 强化“当天最强势板块 + 前排个股”的提取稳定性。
- 提高同花顺/东方财富复盘源解析质量。
- 对 TickFlow 行业/主题归类做更多映射校正。
- 增加报告质量检查：如果强势板块缺少前排股票，给出醒目提示。

## 优先级 2：每天一键运行

- 保持 `make report DATE=YYYY-MM-DD` 作为主入口。
- 增加“生成并自动启动预览服务”的命令。
- 增加最近交易日自动判断。
- 增加报告生成日志摘要。

## 优先级 3：HTML 体验

- 继续优化阅读宽度、表格密度和移动端表现。
- 增加目录导航。
- 增加来源可信度说明。
- 增加模块折叠/打印样式。

## 优先级 4：自选股与 OCR

- 优化同花顺自选股导入。
- 改进截图 OCR 识别准确率。
- 将自选股与当天强势板块做更明确的匹配。
- 继续保持自选股模块默认关闭。

## 优先级 5：部署与自动化

- GitHub Actions：自动测试、lint、Docker build。
- 可选：Docker Hub 或 GHCR 镜像发布。
- 可选：定时任务生成报告。
- 可选：报告归档和索引页面。

## 暂不优先

- 多用户登录系统。
- 在线 SaaS 化。
- 复杂权限体系。
- 自动交易或下单功能。

本项目是复盘工具，不是交易执行系统。报告中的观点和仓位建议都应作为观察纪律，不构成投资建议。
